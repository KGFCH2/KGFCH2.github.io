## 🔎 Overview

A modern, fast, and responsive personal portfolio built with React, TypeScript, Vite, and Tailwind CSS. Fully static — deploy to GitHub Pages or any static host.

## ✨ Highlights

- 🎨 **Dark-first responsive UI** with subtle animations
- 🛡️ **Type-safe** via TypeScript
- 🎯 **Easy theming** with Tailwind utilities
- 🎭 **Smooth transitions** using Framer Motion
- ⚡ **High Performance** — optimized build, Lighthouse-ready
- 🔧 **100% Static** — no backend needed

## 🚀 Quick Start

Prerequisites: Node.js 16+ and npm

Clone and install:

```bash
git clone https://github.com/KGFCH2/KGFCH2.github.io.git
cd Port
npm install
```

Run dev server:

```bash
npm run dev
```

Build and deploy:

```bash
npm run build
npm run preview
npm run deploy
```

## 🎨 Customize

- **Personal Info**: Edit `src/components/Contact.tsx`
- **Projects**: Edit `src/components/Projects.tsx`
- **Skills**: Edit `src/components/Skills.tsx`
- **Colors/Fonts**: Edit `tailwind.config.js` and `src/index.css`
- **Resume PDF**: Placed in `public/Babin_Bid_Resume.pdf` for the "Download Resume" CTA

## 🌐 GitHub Pages Deployment

1. Keep `vite.config.ts` with `base: '/'` for user site
2. Ensure GitHub Pages is set to deploy from `gh-pages` branch (root)
3. Run `npm run deploy`
